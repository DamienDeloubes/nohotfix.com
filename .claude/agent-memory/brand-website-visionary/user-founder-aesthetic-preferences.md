Damien is the founder/builder of NoHotfix, a B2B SaaS release governance tool. He owns both product and marketing decisions. Email: deloubesd@gmail.com.

**Stated aesthetic preferences:**
- "Clean, modern, slightly minimal" — his exact brief for the marketing site
- Orange as the primary color — this was a deliberate rebrand away from blue
- The product should feel serious and precise, not playful or startup-quirky

**Design sensibility — UPDATED after authorized rebrand brief (2026-05-28):**
- Light-first as the new primary theme (Cloudflare-leaning) with dark as a co-equal peer, auto-switched via `prefers-color-scheme`
- Orange kept and carried forward — refined for light-mode legibility (darker interactive value), unchanged in dark mode
- Cloudflare and Scalora cited by founder as key inspiration references
- Prefers restrained glass (Linear-discipline) — glass now restricted to dark mode only
- No blue (retired); no violet-base dark surfaces (retiring `#0D0920` dark-dominant palette)
- Values product UI as the primary visual content — no abstract illustrations
- Engineering-first voice: precise, grounded, confident — never hype

**Product anchors to know:**
- NoHotfix is a release governance tool: enforced artifact collection + go/no-go gate + run immutability = "Ship it once."
- Tagline "Ship it once." is open to evolve but proposal recommends keeping it — strong and product-accurate
- The fire-in-the-o wordmark CONCEPT is kept; execution gets a light-mode refinement pass (not a full redesign)
- Authorized full rebrand underway — `brand-identity.md` will be updated AFTER proposal sign-off

**What to avoid recommending:**
- Any reintroduction of blue
- Violet-base dark surfaces (the `#0D0920` palette era is ending)
- Fire gradient used decoratively outside the logo glyph
- Heavy glassmorphism; glass is now dark-mode-only
- Character illustrations or mascots
- Dark-dominant as the default framing

See [[project-nohotfix-website-vision]] for the full website direction, [[reference-nohotfix-brand]] for color tokens.
